feat(dashboard): revamp hero section with AI insights and harden cycle generation

This commit introduces significant improvements to the dashboard UI, data integrity, and documentation.

UI & Layout:
- Transformed the Dashboard Hero section into a two-column layout: shifted the Phase Ring to the left and introduced a new ambient RedDot.ai Engine insights panel on the right.
- Redesigned the "Log Today" and "Insights" cards with a horizontal checklist format, glassmorphic hover effects, and a new mini stats preview showing Avg Mood and Avg Sleep.
- Enhanced DayDetail popup with subtle red gradient styling and a "Period" tag to visually indicate period days.

Data Integrity & Auth:
- Implemented recalculateCycles() in data.ts to replace rigid cycle creation. It dynamically groups contiguous period days into cycles, fuzzy-matches existing cycles to preserve manual user notes (within +/- 5 days), and automatically garbage collects stale "ghost" cycles.
- Fixed a timezone-drift bug in cycle.ts phase calculation by performing Math.round on local midnight timestamps instead of drifting UTC timestamps.
- Protected OnboardingPage with AuthGuard to prevent UI errors and seamlessly redirect users to the login screen upon session timeout.

Documentation:
- Updated 04_DATA_MODEL.md to formally document the client-side cycle calculation, fuzzy matching, and garbage collection algorithm.
- Updated 06_PAGES_AND_FLOWS.md to reflect the new AI report in the Dashboard hero section and the Onboarding AuthGuard logic.
- Logged Session 9 updates to walkthrough_latest.md.
